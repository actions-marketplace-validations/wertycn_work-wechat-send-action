### Github Action 企业微信消息发送插件

通过企业微信自定义应用发送通知，支持执行失败后发送失败通知、自定义模板

**支持消息类型**：
* <font color=\"info\">文本</font>
* <font color=\"info\">markdown</font>

**支持发送模式**：
* <font color=\"warning\">调用时发送</font>
* <font color=\"warning\">完成时发送</font>
